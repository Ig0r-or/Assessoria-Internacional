# Aba Oportunidades CGRI - Assessoria Internacional IFSULDEMINAS 🌍✈️
📝 Sobre o Projeto
Este projeto foi desenvolvido durante o meu estágio na Assessoria Internacional (CGRI) do IFSULDEMINAS. Fui encarregado de reestruturar e criar a estilização da aba de "Oportunidades CGRI" no portal do setor.

O objetivo principal foi construir uma interface limpa e intuitiva para melhorar a visualização das oportunidades de estágios e intercâmbios internacionais para os alunos da instituição.

🛠️ Tecnologias Utilizadas
HTML5: Estruturação semântica do conteúdo.

CSS3: Estilização, layout e responsividade para garantir uma boa experiência de leitura.

🎯 O que foi feito
Organização das informações de vagas de forma mais acessível.

Criação de uma hierarquia visual clara para facilitar a navegação dos estudantes.

Código limpo e de fácil manutenção para futuras atualizações da equipe.
